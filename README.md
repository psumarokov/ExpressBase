ExpressBase
=========
Basic Express.js + MongoDB server for fast kick off

Requirements
---------
- Node.js v0.12.2
- MongoDB v3.0.2

Running from CLI
---------
    cd "project_directory"
    npm install
    mongod --dbpath "project_directory/database"
    node bin/www

Features
--------
- Express routing
- HTTP CRUD
- File upload
- Integration with MongoDB via Mongoose
- Promises
